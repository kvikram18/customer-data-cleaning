# Customer Data Cleaning & Preprocessing

## 📌 Project Overview

This project demonstrates an end-to-end data cleaning and preprocessing workflow using Python. The objective was to transform a raw customer dataset into a clean, consistent, and analysis-ready dataset by addressing common data quality issues.

---

## 🎯 Objectives

- Handle missing values
- Correct inconsistent data types
- Standardize categorical values
- Remove duplicate records
- Detect and remove outliers
- Validate the cleaned dataset

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy

---

## 📋 Data Cleaning Steps

### 1. Data Inspection
- Examined dataset structure
- Checked data types
- Identified missing values
- Generated descriptive statistics

### 2. Missing Value Treatment
- Removed records with missing Customer_ID
- Filled numerical columns using the median
- Filled categorical columns using the mode

### 3. Data Type Conversion
- Converted Age to numeric format
- Converted Date columns to datetime format

### 4. Standardization
- Corrected inconsistent text values
- Standardized categorical data

### 5. Duplicate Handling
- Identified duplicate records
- Removed duplicate entries

### 6. Outlier Detection
- Visualized outliers using boxplots
- Applied Z-score method for outlier removal

### 7. Validation
- Verified missing values
- Confirmed correct data types
- Validated cleaned dataset
